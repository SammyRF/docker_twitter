# docker_twitter

A mini twitter backend service with features:
* create user/user profile
* follow/unfollow user
* newsfeed
* create tweet
* like tweet/comment
* comment tweet/comment
* notification

based on 
* Python
* Django/Django rest-framework
* Mysql
* AWS S3
* redis
* memcached
* rabbit MQ
* HBase

All services are running in docker
